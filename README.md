# clidongle-web

Browser **WebSerial** UI for **CLI Dongle** — a USB dongle (Raspberry Pi Pico 2 W /
Waveshare RP2350-USB-A) that sits inline between a keyboard and a computer, adding key
remapping, macros, an encrypted password manager, notes, and Gemini AI.

Open the hosted page over HTTPS in **Chrome or Edge**, click **Connect**, and choose the
dongle's serial port. Everything runs locally in your browser talking to the dongle
plugged into *your own* computer's USB — no data leaves the machine.

Firmware source is maintained separately.
